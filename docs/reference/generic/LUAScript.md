# Configuration



## /settings/lua

`/settings/lua`

**LUA SCRIPT SECTION**

Section for the LUAScripts module.




**Sample**::

```
# LUA SCRIPT SECTION
# Section for the LUAScripts module.
[/settings/lua]

```




## /settings/lua/scripts

`/settings/lua/scripts`

**LUA SCRIPTS SECTION**

A list of scripts available to run from the LuaSCript module.




**Sample**::

```
# LUA SCRIPTS SECTION
# A list of scripts available to run from the LuaSCript module.
[/settings/lua/scripts]

```
